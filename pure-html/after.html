Tailwind CSS

Tailwind CSS is an open source CSS framework. The main feature of this library is that, unlike other CSS frameworks, it does not provide a series of predefined classes for elements.

Vim

Vim or ViImproved - the ubiquitous text editor Vim is a highly configurable text editor built to make creating and changing any kind of text very efficient. It is included as "vi" with most UNIX systems and with Apple OS X.
